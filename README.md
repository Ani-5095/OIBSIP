Java Core Projects: Logic & Interfaces
A collection of functional Java applications demonstrating core programming principles, object-oriented design, and interactive console-based logic. This repository features two primary modules: a Number Guessing Game and an ATM Interface.

Tech Stack
Language: Java (JDK 17+ recommended)

Environment: Cross-platform (macOS, Windows, Linux)

Concepts: Object-Oriented Programming (OOP), Exception Handling, Collections Framework.

Projects Included
1. ATM Interface
A simulation of a real-world Automated Teller Machine. This project focuses on secure logic and state management.

Key Features:

User Authentication: Secure PIN-based login.

Transaction History: Keeps track of every deposit and withdrawal.

Balance Inquiry: Real-time updates on account status.

Withdrawal/Deposit: Validated logic to prevent overdrafts.

2. Number Guessing Game
An interactive game designed to test logic and user input handling.

Key Features:

Difficulty Scaling: Limits on the number of attempts.

Score Tracking: Points based on how quickly the number is identified.

Dynamic Feedback: Hints like "Too High" or "Too Low" to guide the user.
